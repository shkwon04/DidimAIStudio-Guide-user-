```markdown
## 에이전트 관리 페이지

에이전트를 효율적으로 관리할 수 있는 전용 페이지가 새롭게 추가되었습니다.

### 주요 기능
- 에이전트 **목록 조회** 및 **상세 정보** 확인을 위한 UI 레이아웃을 구축하였습니다.
- 목록 조회 기능은 `useQuery`를 활용하여 API와 연동하였으며, 에이전트 생성, 수정, 삭제 기능은 `useMutation`으로 구현하였습니다.
- `AgentForm` 컴포넌트를 분리하여 코드 재사용성을 높였으며, 입력값 검증(Validation) 로직을 추가하였습니다.

### UI/UX 개선
- 로딩 상태 시 **스켈레톤 UI**를 표시하여 사용자 경험을 개선하였으며, 성공 및 에러 상황에 대한 **토스트 알림**을 적용하였습니다.
- `shadcn/ui` 기반의 스타일을 적용하였으며, 다양한 화면 크기를 지원하기 위해 반응형 디자인을 구현하였습니다.

### 기술적 개선
- 타입 정의(Agent, ModelConfig)을 추가하여 코드의 가독성을 향상하였으며, 전체 폴더 구조를 정리하였습니다.
```
