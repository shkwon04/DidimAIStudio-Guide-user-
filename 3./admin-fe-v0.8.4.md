## 에이전트 관리 페이지

이번 업데이트에서는 에이전트를 효율적으로 관리할 수 있는 **전용 페이지**가 새롭게 추가되었습니다. 사용자분들이 더욱 편리하게 에이전트를 관리할 수 있도록 다양한 기능을 구현하였습니다.

### 주요 변경 사항

**에이전트 목록 및 상세 UI 레이아웃**이 구축되어, 전체 에이전트를 한눈에 확인할 수 있게 되었습니다. 각 에이전트의 상세 정보도 손쉽게 조회할 수 있어 더욱 편리합니다.

또한, 목록 조회 API와의 연동을 통해 **에이전트 생성, 수정, 삭제** 기능을 지원합니다. 이를 위해 `useQuery`와 `useMutation`을 활용하여 데이터 관리를 효율적으로 처리하였습니다. 

에이전트 관련 작업을 위한 **AgentForm 컴포넌트**를 분리하여 관리의 용이성을 높였으며, 입력 데이터에 대한 **Validation 로직**도 추가하여 사용자 경험을 개선했습니다.

로딩 중에는 **로딩 스켈레톤**이 표시되어 사용자가 대기하는 동안 시각적인 피드백을 제공하며, 작업 완료 시에는 성공 및 에러 상황에 맞는 **토스트 메시지**가 표시되어 보다 직관적인 사용이 가능합니다.

### UI/UX 개선

이번 업데이트에서는 **shadcn/ui** 기반의 스타일을 적용하여 전반적인 디자인을 개선하였고, 반응형 디자인을 통해 다양한 화면 크기에서도 안정적인 UI를 제공할 수 있도록 하였습니다. 

또한, 타입 정의를 추가하고 폴더 구조를 정리하여 코드의 가독성과 유지보수성을 한층 높였습니다.
