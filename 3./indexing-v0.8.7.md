## Reranker 마이그레이션 및 BM25 OpenSearch 전환

이번 업데이트에서는 Reranker 서비스를 OpenSearch BM25로 마이그레이션하며, 
여러 가지 기능 개선과 코드 리팩토링을 통해 전반적인 성능을 향상시켰습니다. 
사용자 여러분께 더 나은 서비스 환경을 제공하기 위해 최선을 다했습니다.

### 주요 변경 사항

**Elasticsearch에서 OpenSearch BM25로의 마이그레이션**이 완료되었습니다. 
이로 인해 검색 성능이 더욱 향상되었으며, 보다 정확한 결과를 제공할 수 있게 되었습니다. 

또한, Reranker 원격 서비스의 인증 및 테스트 환경이 개선되었습니다. 
이와 함께 로컬 테스트 환경 설정도 더욱 수월해졌으며, 인증 우회 기능을 추가하여 
개발자들이 보다 쉽게 작업할 수 있도록 지원합니다.

이번 업데이트에서는 코드 컨벤션을 전면적으로 리팩토링하여 
가독성과 유지보수성을 높였습니다. 

**임베딩 롤백 완료 시 SSE 알림 발행 기능**이 추가되어, 
실시간으로 작업 상태를 확인할 수 있게 되었습니다. 

마지막으로, Milvus 연결 최적화 작업 및 엔티티 배치 삭제 기능이 구현되어 
데이터 관리가 한층 원활해졌습니다. 
더불어 미사용 라이브러리도 정리하여, 코드의 경량화를 이뤘습니다. 
이로 인해 프로젝트의 불필요한 복잡성이 줄어들었습니다.

이번 업데이트를 통해 더욱 향상된 기능을 경험하시길 바랍니다.
