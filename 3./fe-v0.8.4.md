## 에이전트 관리 페이지

이번 업데이트에서는 **에이전트 관리 페이지**가 새롭게 제작되었습니다. 사용자 여러분이 에이전트를 보다 쉽게 관리할 수 있도록 다양한 기능을 추가하였으며, UI/UX 개선에도 신경을 썼습니다.

### 주요 변경 사항

**에이전트 목록 및 상세 정보**를 한눈에 확인할 수 있는 UI 레이아웃이 구축되었습니다. 이를 통해 사용자는 각 에이전트의 정보를 직관적으로 조회할 수 있습니다. 또한, 목록 조회 API와의 연동을 통해 데이터의 최신성을 유지하고, 에이전트의 생성, 수정, 삭제 기능을 통해 필요한 작업을 손쉽게 수행할 수 있습니다.

에이전트 관리에 필요한 **AgentForm 컴포넌트**가 분리되어, 유효성 검사 로직도 추가되었습니다. 이를 통해 사용자는 더욱 안정적으로 데이터를 입력할 수 있습니다.

추가적으로, 화면 로딩 시 사용자 경험을 고려하여 **로딩 스켈레톤**과 작업 완료 시 **성공/실패 알림**을 표시하는 토스트 메시지가 적용되었습니다. 이러한 요소들은 사용자에게 보다 원활한 피드백을 제공하여, 에이전트 관리의 효율성을 높여줍니다.

### UI/UX 개선

이번 업데이트에서는 **shadcn/ui** 기반의 스타일이 적용되어, 더욱 깔끔하고 현대적인 디자인을 구현하였습니다. 또한, 화면 크기에 따라 자동으로 레이아웃이 조정되는 **반응형 디자인**을 적용하여 다양한 기기에서도 최적의 사용자 경험을 제공합니다.

마지막으로, 타입 정의(Agent, ModelConfig)를 추가하고 폴더 구조를 정리하여 코드의 관리성과 가독성을 향상시켰습니다. 

이처럼 여러분의 편리한 에이전트 관리 경험을 위해 여러 개선 사항을 반영하였으니, 많은 이용 부탁드립니다.
