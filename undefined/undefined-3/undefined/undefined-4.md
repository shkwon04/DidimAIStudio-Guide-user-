# 시나리오 관리

## 1. 시나리오 관리 페이지 개요

시나리오 관리 페이지는 AI 에이전트 실행 흐름을 정의하는“작업 템플릿 단위 관리 시스템”입니다.

각 시나리오는 하나의 AI 실행 구조(Flow / Agent Workflow)를 의미합니다.

## 2. 검색 기능

***

### &#x20;이름 검색

* 시나리오 이름 기준 검색 기능
* 입력한 키워드와 일치하는 시나리오 필터링

&#x20;예:

* “Deep Agent”
* “보고서”

&#x20;빠르게 특정 시나리오 탐색 가능



시나리오 생성부분은 사용자에서 워크플로우로 진행할수있습니다.
